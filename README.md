Las features del Front-End en tanto código son:
1. Pantallas de autentificación: Login, Register, ForgotPassword, ResetPassword.
2. Protección de rutas sensibles.
3. Uso de hooks como useForm para manejo de datos de formulario y useSearchTerms para filtro por texto.
4. Uso de contexto global para permitir el acceso a rutas sensibles.
5. Fetching de datos pertenecientes a cada usuario.

Lo mas desafiante del proyecto fue hacer la conexión perfecta con el back, mandándole los datos necesarios de la manera correcta.
